Estrutura

Nós nos deparamos com todos os tipos de documentos todos os dias de nossas vidas. Jornais, formulários de seguros, catálogos de lojas... a lista continua.

Muitas páginas da web agem como versões eletrônicas desses documentos. Por exemplo, jornais mostram as mesmas histórias impressas que em sites; você pode solicitar seguro pela web; e lojas têm catálogos online e sistema de comércio eletrônico.

Em todos os tipos de documentos, a estrutura é muito importante para ajudar os leitores a entender as mensagens que você está tentando transmitir e a navegar pelo documento. Então, para aprender a escrever páginas da web, é muito importante entender como estruturar documentos. Neste capítulo, você irá:

  • Ver como o HTML descreve a estrutura de uma página da web
  • Aprender como tags ou elementos são adicionados ao seu documento
  • Escrever sua primeira página web

Como as páginas usam a estrutura

Pense nas histórias que você lê em um jornal: para cada história, haverá uma manchete, algum texto e possívelmente algumas imagens. Se o artigo for longo, pode haver subtítulos que dividem a história em seções separadas ou citações daqueles envolvidos. A estrutura ajuda os leitores a entender as histórias no jornal.

A estrutura é muito similar quando uma notícia é visualizada online (embora também possa apresentar áudio e vídeo). Isso é ilustrado à direita com uma cópia de um jornal ao lado do artigo correspondente em seu site.

Agora pense em um tipo bem diferente de documento - um formulário de seguro. Formulários de seguro geralmente têm títulos para seções diferentes, e cada seção contém uma lista de perguntas com áreas para você preencher detalhes ou caixas de seleção para marcar. Novamente, a estrutura é bem parecida online.

Estruturação de documentos word

O uso de títulos e subtítulos em qualquer documento frequentemente reflete uma hierarquia de informações. Por exemplo, um documento pode começar com um título grande, seguido por uma introdução ou as informações mais importantes.

Isso pode ser expandido em subtítulos mais abaixo na página. Ao usar um processador para criar um documento, separamos o texto para dar-lhe estrutura. Cada tópico pode ter um novo parágrafo, e cada seção pode ter um título para descrever o que ele cobre.

À direita, você pode ver um documento simples no Microsoft Word. Os diferentes níveis de título, são mostrados na caixa suspensa. Se você usa o Word regularmente, você também pode ter usado a barra de ferramentas ou paleta de formatação para fazer isso.

Anteriormente você viu como a estrutura foi adicionada a um documento do Word para torná-lo mais fácil de ler. Usamos a estrutura da mesma forma ao escrever páginas da web.

HTML descreve a estrutura das páginas

Na janela do navegador, você pode ver uma página da web que apresenta o mesmo conteúdo do documento do Word que você viu na página 18. Para descrever a estrutura de uma página da web, adicionamos código às palavras que queremos que apareçam na página.

Você pode ver o código HTML para esta página abaixo. Não se preocupe com o que o código significa ainda. Começaremos a olhar para ele em mais detalhes na próxima página. Note que o código HTML está em azul, e o texto que você vê na tela está em preto.

O código HTML (em azul) é composto de caracteres que ficam dentro de colchetes angulares - eles são chamados de elementos HTML. Os elementos são geralmente compostos de duas tags: uma tag de abertura e uma tag de fechamento. (A tag de fechamento tem uma barra extra.) Cada elemento HTML diz ao navegador algo sobre as informações que ficam entre suas tags de abertura e fechamento.

HTML usa elementos para descrever a estrutura da página

Vamos olhar mais de perto o código visto anteriormente. Há vários elementos diferentes. Cada elemento tem uma tag de abertura e uma tag de fechamento.

As tags agem como contêineres. Elas dizem algo sobre as informações que ficam entre suas tags de abertura e fechamento.

Atributos nos contam mais sobre o elemento

Atributos fornecem informações adicionais sobre o conteúdo de um elemento. Eles aparecem na tag de abertura do elemento e são compostos de duas partes: um nome e um valor, separados por um sinal de igual.

O nome do atributo indica que tipo de informação extra você está fornecendo sobre o conteúdo do elemento. Ele deve ser escrito em letras minúsculas.

O valor é a informação ou configuração para o atributo. Ele deve ser colocado entre aspas duplas. Atributos diferentes podem ter valores diferentes.

Aqui, um atributo chamado lang é usado para indicar o idioma usado nesse elemento. O valor deste atributo nesta página especifica que le está em inglês dos Estados Unidos.

A maioria dos atributos só pode ser usada em certos elementos, embora alguns atributos (como lang) possam aparecer em qualquer elemento.

A maioria dos valores de atributos são predefinidos ou seguem um formato estipulado. Veremos os valores permitidos à medida que introduzimos cada novo atributo.

O valor do atributo lang é uma maneira abreviada de especificar qual idioma é usado dentro do elemento que todos os navegadores entendem.

O HTML5 permite que você use nomes de atributos em letras maiúsculas e omita as aspas, mas isso não é recomendado.

Corpo, cabeça e título

Corpo

Você conheceu o elemento body no primeiro exemplo que criamos. Tudo dentro desse elemento é mostrado dentro da janela principal do navegador.

Cabeça

Antes do elemento body, você frequentemente verá um elemento head. Ele contém informações sobre a página (em vez de informações mostradas na parte principal da janela do navegador, destacada em azul na imagem abaixo. Você normalmente encontrará um elemento title dentro do elemento head.

Título

O conteúdo do elemento title é exibido na parte superior do navegador, acima de onde você digita normalmente o URL da página que deseja visitar, ou na guia dessa página (se o seu navegador usar guias para permitir que você visualize várias páginas ao mesmo tempo).

Você pode saber que HTML significa HyperText Markup Language (Linguagem de marcação de hipertexto). A parte HyperText se refere ao fato de que HTML permite que você crie links que permitem que os visitantes se movam de uma página para outra de forma rápida e fácil. Uma linguagem de marcação permite que você anote texto, e essas anotações fornecem significado adicional ao conteúdo de um documento. Se você pensar em uma página web, adicionamos código ao redor do texto original que queremos exibir e o navegador então usa o código para exibir a página corretamente. Então, as tags que adicionamos são a marcação.
