Texto

Ao criar uma página da web, você adiciona tags (conhecidas como marcação) ao conteúdo da página. Essas tags fornecem significado extra e permitem que os navegadores mostrem ao usuário a estrutura apropriada para a página.

Neste capítulo, focamos em como adicionar marcação ao texto que aparece nas páginas. Você aprenderá sobre:

  • Marcação estrutural: os elementos que você pode usar para descrever títulos e parágrafos.
  • Marcação semântica: que fornece informações extras; como onde a ênfase é colocada em uma frase, que algo que você escreveu é uma citação (e quem o disse), o significado de siglas e assim por diante.

Títulos

<h1>, <h2>, <h3>, <h4>, <h5> e <h6>

O HTML tem seis níveis de títulos:

<h1> é usado para títulos principais

<h2> é usado para subtítulos

Se houver mais seções sob os subtítulos, o elemento <h3> será usado, e assim por diante...

Os navegadores exibem o conteúdo dos títulos em tamanhos diferentes. O conteúdo de um elemento <h1> é o maior, e o conteúdo de um elemento <h6> é o menor. O tamanho exato em que cada navegador mostra os títulos podem variar um pouco. Os usuários também podem ajustar o tamanho do texto em seus navegadores. Você verá como controlar o tamanho do texto, sua cor e as fontes usadas quando chegarmos ao CSS.

Parágrafos

p

Para criar um parágrafo, coloque as palavras que compõem o parágrafo entre uma tag de abertura <p> e uma tag de fechamento </p>.

Por padrão, um navegador mostrará cada parágrafo em uma nova linha com algum espaço entre ele e qualquer parágrafo subsequente.

Negrito e itálico

<b>

Ao colocar palavras entre as tags <b> e </b>, podemos fazer com que os caracteres apareçam em negrito.

O elemento <b> também representa uma seção de texto que seria apresentada de uma maneira visualmente diferente (por exemplo, palavras-chave de um parágrafo), embora o uso do elemento <b> não implique nenhum significado adicional.

<i>

Ao colocar palavras entre as tags <i> e </i>, podemos fazer com que os caracteres apareçam em itálico.

O elemento <i> também representa uma seção de texto que seria dita de uma maneira diferente do conteúdo ao redor - como termos técnicos, nomes de navios, palavras estrangeiras, pensamentos ou outros termos que normalmente seriam colocados em itálico.

Sobrescrito e subscrito

<sup>

O elemento <sup> é usado para conter caracteres que devem ser sobrescritos, como sufixos de datas ou conceitos matemáticos, como elevar um número a uma potência como 22.

<sub>

O elemento <sub> é usado para conter caracteres que devem ser subscritos. É comumente usado com notas de rodapé ou fórmulas químicas como H2O.

Espaço em branco

Para tornar o código mais fácil de ler, os autores de páginas da web geralmente adicionam espaços extras ou iniciam alguns elementos em novas linhas.

Quando o navegador encontra um ou mais espaços um ao lado do outro, ele exibe apenas um espaço. Da mesma forma, se ele encontrar uma quebra de linha, ele a trata como um único espaço também. Isso é conhecido como colápso de espaço em branco.

Você verá frequentemente que os autores de páginas da web aproveitam o colápso de espaços em brando para recuar seu código, a fim de torná-lo mais fácil de ler.

Quebras de linhas e

<b>

Como você já viu, o navegador mostrará automaticamente cada novo parágrafo ou título em uma nova linha. Mas se você quiser adicionar uma quebra de linha no meio de um parágrafo, você pode usar a tag de quebra de linha <b>.

<hr>

Para criar uma pausa entre temas, como uma mudança de tópico em um livro ou uma nova cena em uma peça, você pode adicionar uma régua horizontal entre as seções usando a tag <hr>.

Existem alguns elementos que não têm nenhuma palavra entre a tag de abertura e fechamento. Eles são conhecidos como elementos vazios e são escritos de forma diferente.

Um elemento vazio geralmente tem somente uma tag. Antes do colchete angular de fechamento de um elemento vazio, geralmente haverá um espaço e um caractere de barra.
